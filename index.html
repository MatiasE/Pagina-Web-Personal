Introduccion a CSS:

¿Que es CSS? CSS, acrónimo de Cascading Style Sheets(hojas de estilo en cascada), sirve para darle presentacion a la pagina web. Su sintaxis es la siguiente, un conjunto de reglas CSS consta de un selector y un bloque de declaración: Selector {Propiedad: Valor;}. Etiquetas basicas:

p { color: red; text-align: center; }
